# 千牛聊天记录导出器

一款专为千牛网页版设计的油猴脚本，可自动导出所有客户聊天记录，支持自定义设置、实时进度显示，操作简单、稳定可靠，彻底解决手动导出繁琐、批量下载拦截等问题。

## 📌 核心功能

- ⚙️ 自定义设置：可设置导出客户数量、切换聊天等待时间，适配不同网络速度

- 📦 批量导出：自动遍历左侧所有客户，模拟点击切换，抓取对应聊天记录

- 压缩打包：所有客户聊天记录按用户分类（每人1个JSON文件），统一打包为ZIP，单次下载不拦截

- 📊 实时进度：导出过程显示进度条、完成百分比、预计剩余时间，直观了解进度

## 🔧 环境要求

- 浏览器：Chrome / Edge 最新版（其他Chromium内核浏览器也可兼容）

- 插件：Tampermonkey（油猴）最新版（用于运行脚本）

- 网页：千牛网页版（https://myseller.taobao.com/home.htm/app-customer-service/toolpage/Message)

## 📥 安装步骤

1. 安装 Tampermonkey 插件

   - Chrome：应用商店搜索「Tampermonkey」安装

   - Edge：扩展商店搜索「Tampermonkey」安装

2. 打开 Tampermonkey 插件，点击「添加新脚本」

3. 全选删除默认生成的脚本代码

4. 复制本项目的完整脚本代码，粘贴到编辑框中

5. 按 `Ctrl+S` 保存脚本，脚本自动启用

6. 打开千牛网页版聊天页面，刷新页面，即可看到导出按钮

## 📝 使用方法

1. 打开千牛客服聊天记录查询

2. 找到的「导出聊天记录」绿色按钮，点击按钮

3. 弹出设置弹窗（可按住弹窗顶部拖动位置），设置导出参数：     

   - 最多导出客户数：设置需要导出的客户数量（默认200，填9999可导出全部）

   - 切换等待时间(ms)：切换客户后等待聊天加载的时间（默认600ms，网络慢可调至800-1000ms）

4. 点击「开始导出」，脚本自动执行：

5. 解压ZIP压缩包，每个客户对应1个JSON文件，包含该客户的所有聊天记录

## 📊 导出文件说明

### ZIP文件名格式

`千牛聊天记录_日期.zip`（日期格式：年-月-日，如：千牛聊天记录_2026-04-07.zip）

### JSON文件格式（每个客户1个文件）

```json
{
  "user": "客户昵称/ID",          // 聊天对象名称
  "exportTime": "导出时间",       // 脚本导出的时间（本地时间）
  "count": 聊天记录条数,          // 该客户的聊天记录总条数
  "chats": [                      // 聊天记录数组
    {
      "date": "聊天日期",         // 聊天所属日期
      "time": "聊天时间",         // 单条消息的具体时间
      "sender": "发送人",         // 消息发送者（客户/商家）
      "text": "消息内容",         // 文本消息内容（优先取完整标题，避免内容截断）
      "imgUrl": "图片链接"        // 消息中的图片链接（无图片则为空字符串）
    }
  ]
}
```
